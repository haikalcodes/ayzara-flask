## ğŸ‘¥ Untuk Admin

### 1. Login Admin
*   Buka aplikasi.
*   Masukkan username dan password admin Anda.
*   Klik tombol **Login**.

### 2. Dashboard Monitoring
Setelah login, Anda akan masuk ke halaman Dashboard.
*   **Statistik Harian**: Melihat jumlah packing hari ini (Total & Selesai).
*   **Live Monitoring**: Memantau kamera CCTV secara real-time.
*   **Aktivitas Terkini**: Melihat log aktivitas packing yang baru saja terjadi.

### 3. Manajemen Pegawai (Menu Team)
Masuk ke menu **Team** untuk mengelola data pegawai.
*   **Tambah Pegawai**:
    1. Klik tombol **"Tambah Anggota"**.
    2. Isi **Nama**, **Jabatan**, **No. Telepon** (Wajib, digunakan sebagai password login pegawai), dan upload foto.
    3. Klik **Simpan**.
*   **Edit Pegawai**: Klik ikon pensil pada kartu pegawai untuk mengubah data.
*   **Nonaktifkan Pegawai**: Klik tombol user untuk menonaktifkan akses pegawai tanpa menghapus datanya.
*   **Hapus Pegawai**: Klik ikon sampah untuk menghapus data pegawai permanen.

### 4. Melihat Rekaman (Menu Videos)
Masuk ke menu **Videos** untuk melihat riwayat rekaman packing.
*   **Filter**: Cari berdasarkan Resi, Nama Pegawai, atau Platform (Shopee, Tokopedia, TikTok, dll).
*   **Putar Video**: Klik thumbnail video untuk memutar rekaman.

## ğŸ‘· Untuk Pegawai

### 1. Login Pegawai
*   **Username**: Nama Anda (sesuai yang didaftarkan Admin).
*   **Password**: Nomor Telepon Anda.

### 2. Dashboard Pegawai
Setelah login, Anda akan melihat statistik kinerja Anda sendiri:
*   Total paket yang dipacking hari ini.
*   Rata-rata durasi packing.
*   Riwayat packing terakhir Anda.

### 3. Melakukan Rekaman Packing (Menu Rekam)
Ini adalah fitur utama untuk merekam proses packing.
1.  **Siapkan Barang**: Letakkan barang di meja packing.
2.  **Scan/Input Resi**: Scan barcode resi atau ketik nomor resi di kolom yang tersedia.
3.  **Mulai Rekam**:
    *   Pastikan kamera menampilkan gambar yang jelas.
    *   Klik tombol **"Mulai Rekam"** (atau tekan Enter jika menggunakan scanner).
4.  **Proses Packing**: Lakukan packing seperti biasa di depan kamera.
5.  **Selesai**:
    *   Klik tombol **"Selesai"** (Warna Merah) setelah packing selesai.
    *   Video akan otomatis tersimpan.

### Tips:
*   Pastikan pencahayaan cukup agar wajah dan resi terlihat jelas.
*   Jika terjadi kesalahan, Anda bisa menekan tombol **Cancel** sebelum menyimpan.
